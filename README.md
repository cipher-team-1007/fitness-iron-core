# Gravity Fitness Gym

A premium web application for **Gravity Fitness Gym** in Patna — the hardcore training ground for strength athletes and powerlifters. 

## Features
- **Modern Dark Aesthetic**: Styled with premium typography (Anton & Inter) and dynamic cinematic film grain overlays.
- **Section Layouts**: Includes Hero banner, training Stats, Programs details, Coaches profiles, Facility Gallery (using high-quality photography), Schedule tables, Membership packages, and Contact/Location forms.
- **Responsive Navigation**: Smooth mobile drawer menu and custom layout sections built using Framer Motion animations.
- **Tech Stack**: React 18, TypeScript, Tailwind CSS, Framer Motion, Lucide icons, Vite, and Supabase client integration.
